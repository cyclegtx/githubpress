## 安装
`npm install githubpress -g`

```
使用方法: githubpress <命令>
	命令:
		init      初始化博客
		repos     管理Repositories的列表
		getall    根据repos.json获取README新建文章
		generate  生成静态文件
		clean	  清除没有内容的文章
		help      获取帮助
		list      列出所有文章
		get       从github获取README新建文章
		update    从github获取最新README更新文章
		new       新建文章
		author	  添加或者删除github帐号
		push      发布文章到github

命令帮助: githubpress 命令 help
```
